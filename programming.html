Robot Programming Overview

Our robot was programmed using a logic-based algorithm to detect blood clots and release medication when necessary. The programming includes proximity sensors and chemical detectors that allow precise action. The logic code helps control movement, detection, and medication release in real-time. This helps avoid unnecessary drug use and improves safety.
